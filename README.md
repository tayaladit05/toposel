# Toposel Assignment (Simple Project Note)

Hi,

This is my assignment theme for a fashion shopping homepage.
I made it with focus on clean look, mobile support, and simple user flow.

I am writing this in simple language so anyone can understand the project quickly.

## What this project is

This is a custom WordPress theme page for an online clothing store style website.
Main goal was to create a homepage that looks modern and easy to use.

The page has:
- Top message bar (offer line)
- Navbar with logo, menu, search, cart and account icon
- Hero section (big heading + button + image)
- Brand strip (logo row)
- New arrivals product section

## My approach (simple)

I tried to keep things:
- Clean
- Fast
- Easy to edit later

I also made sure:
- It works on phone view
- It works on desktop view
- Menu opens on mobile with 3-line button

## Files and what they do

- functions.php
Main settings and helper functions are here.
It also loads CSS and JS files.

- header.php
Top area of page (announcement + navbar) is here.

- front-page.php
Homepage content is here (hero, brands, products).

- style.css
All design and responsive styling is here.

- assets/js/theme.js
Mobile menu open/close behavior is here.

- footer.php
Footer closing part and WordPress footer call.

- index.php
Simple fallback page template.

## Data/content side

Some text and images are coming from custom fields.
So content can be changed without touching layout code.



## If someone wants to test quickly

1. Activate this theme in WordPress.
2. Open homepage.
3. Check desktop and mobile view.
4. Check mobile menu button (3-line icon).
5. Update content from backend fields and verify UI updates.
